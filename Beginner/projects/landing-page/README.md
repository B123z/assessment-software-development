# Landing Page Project

## Overview
This project is a beginner-level landing page built as part of the SDU Beginner Track.  
The purpose of the project is to practice basic web development skills by creating a clean and responsive webpage.

---

## Project Goals

- Build a simple landing page using HTML and CSS
- Apply responsive design principles
- Organize project files properly

---

## Technologies Used
- HTML5
- CSS3


---

## Features
- Clean and modern landing page layout
- Flexbox-based structure for alignment
- Responsive design for different screen sizes
- Call-to-action buttons
- Local image assets linked using relative paths

---

## How to View the Project
1. Open the `src` folder
2. Open `index.html` in a web browser

No additional setup or installation is required.

---

## Learning Outcome
Through this project, I learned how to:
- Structure a webpage using semantic HTML
- Use Flexbox to arrange page sections
- Make a webpage responsive using media queries
- Fix image linking issues using relative paths
- Document my work clearly
