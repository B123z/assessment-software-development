# Prompt Usage Documentation

This document outlines how prompts were used during the development of the landing page project. The goal is to demonstrate intentional and effective use of GPT to support learning and problem-solving.

---

## Prompt 1: Landing Page Structure
**Prompt used:**  
"Generate a beginner-friendly landing page using HTML and CSS."

**Why this prompt was used:**  
To understand the basic structure of a landing page and identify the key sections required.

**Problem it helped solve:**  
Provided a clear starting point for organizing the HTML layout.

---

## Prompt 2: CSS Flexbox Layout
**Prompt used:**  
"How can I use Flexbox to arrange elements in a landing page layout?"

**Why this prompt was used:**  
To learn how to align elements properly without using floats or complex positioning.

**Problem it helped solve:**  
Helped structure the navigation bar, hero section, and statistics section using Flexbox.

---

## Prompt 3: Responsive Design
**Prompt used:**  
"How do I make a landing page responsive using CSS media queries?"

**Why this prompt was used:**  
To ensure the webpage remains readable on different screen sizes.

**Problem it helped solve:**  
Enabled layout adjustments for mobile and smaller devices.

---

## Prompt 4: Linking Images Correctly
**Prompt used:**  
"How do I link local images using relative paths in HTML?"

**Why this prompt was used:**  
To fix issues caused by using absolute file paths.

**Problem it helped solve:**  
Ensured images display correctly and the project works across different systems and on GitHub.

---

## Prompt 5: Improving UI and Visual Hierarchy
**Prompt used:**  
"How can I improve the visual hierarchy of a landing page using CSS?"

**Why this prompt was used:**  
To make important elements like headings and call-to-action buttons stand out.

**Problem it helped solve:**  
Improved readability, spacing, and overall user experience.

---

## Summary

The prompts used during this project guided development decisions, helped solve technical issues, and improved understanding of core web development concepts. This demonstrates effective use of prompt engineering during the Beginner Track.
